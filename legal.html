This website is a private educational communication platform created for Class 8 students. It is a non-commercial, school-focused project and is not intended for public commercial use.

2. Compliance with Indian Laws

This platform follows applicable Indian laws including:

  • Information Technology Act, 2000
  • Digital Personal Data Protection Act, 2023
  • Protection of Children from Sexual Offences (POCSO) Act

Data is collected solely for:

This platform is intended for minors (under 18 years). Users must have parental or guardian consent before using this platform, in compliance with the Digital Personal Data Protection Act, 2023.

All data is securely stored using protected cloud services. Access is restricted to authenticated users only. Data is not sold, rented, or shared with third parties.
